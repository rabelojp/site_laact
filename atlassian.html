O que é o Atlassian?

Atlassian é uma empresa de software que oferece ferramentas voltadas para o desenvolvimento de software, gerenciamento de projetos e colaboração em equipe. Entre seus produtos mais conhecidos estão o Jira (gestão de tarefas e projetos), Confluence (colaboração e compartilhamento de conhecimento), e Bitbucket (controle de versões para projetos de código).

Criar uma conta e baixar o aplicativo

Crie uma conta

Acesse o site oficial do Canva (https://www.canva.com/) e clique em Criar uma conta, no canto superior esquerdo, aparecerá uma página para selecionar uma forma de criar sua conta.

Escolha como continuar

Selecione uma forma de continuar com a criação da sua conta. Você pode usar o e-mail cadastrado no Google, Facebook, selecionar o e-mail ou continuar de outras formas.

Página Inicial

Após criar sua conta você será redirecionado para a página inicial onde poderá começar a trabalhar com seus designs.

Faça o download do aplicativo (recomendado)

Recomendamos que seja feito o download e a instalação do aplicativo para maior praticidade na hora de trabalhar. Confira abaixo o passo a passo de como baixar e instalar o Canva na sua máquina.

Link para download

Acesse o site oficial do Canva (link: Download Canva) para fazer baixar o setup de instalação do programa. Selecione o sistema operacional e clique em Download Canva e automaticamente o arquivo será baixado.

Inicialize o setup

Abra o arquivo baixado para inicializar o setup, após isso o processo de instalação acontecerá automaticamente.

Faça login pelo navegador

É necessário que o login na sua conta criada seja feito pelo navegador. Ao clicar no botão de Faça login pelo navegador você será redirecionado para uma página web.

Siga as instruções para fazer login e você será direcionado novamente para o aplicativo.

Tudo pronto

Após seguir os passos corretamente, você será direcionado a essa tela onde poderá criar novos arquivos e começar a criar seus designs.
