Visual Studio Code
(editor de código fonte)

O que é o VSCode?

O Visual Studio Code (VsCode) é um editor de código-fonte gratuito e multiplataforma, desenvolvido pela Microsoft. Ele é leve, rápido e altamente personalizável, suportando várias linguagens de programação, como JavaScript, Python, HTML, CSS, entre outras. O VsCode se destaca por suas extensões, que permitem adicionar funcionalidades específicas, como depuração, controle de versão com Git e formatação de código.

Como baixar e instalar

Acesse o site

Acesse o site oficial do VSCode (https://code.visualstudio.com/) e clique em DOWNLOAD, você será redirecionado para outra página onde conseguirá tem acesso aos links de download.

Sistema Operacional

Selecione o seu sistema operacional e, ao clicar, o download será iniciado automaticamente.

Setup

Abra o programa baixado e o assistente de instalação será iniciado.

Complete a instalação

Configure de acordo sua prefêrencia e clique em Instalar. Depois disso, o aplicativo do VSCode ficará disponível para uso.

Comece um projeto

Para começar um projeto, antes de tudo crie uma pasta para salvar os arquivos que forem feitos. É recomendado criar na área de trabalho para que fique fácil de encontrar posteriormente.

Selecione o diretório

Encontre e seleciona a pasta (diretório) que acabou de criar e clique em Selecionar Pasta.

Tudo pronto

Após seguir os passos corretamente, você será direcionado a essa tela onde poderá criar novos arquivos e começar a criar seus projetos.
