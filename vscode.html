Visual Studio Code
(editor de código fonte)

O que é o VSCode?

O Visual Studio Code (VsCode) é um editor de código-fonte gratuito e multiplataforma, desenvolvido pela Microsoft. Ele é leve, rápido e altamente personalizável, suportando várias linguagens de programação, como JavaScript, Python, HTML, CSS, entre outras. O VsCode se destaca por suas extensões, que permitem adicionar funcionalidades específicas, como depuração, controle de versão com Git e formatação de código.

Como baixar e instalar

Acesse o site

Acesse o site oficial do VSCode (https://code.visualstudio.com/) e clique em DOWNLOAD, você será redirecionado para outra página onde conseguirá tem acesso aos links de download.

Sistema Operacional

Selecione o seu sistema operacional e, ao clicar, você será redirecionado para outra página onde deverá selecionar a arquitetura do seu processador (32-bit ou 64-bit no caso do Windows).

Arquitetura do processador

Selecione o seu sistema operacional e, ao clicar, você será redirecionado para outra página onde deverá selecionar a arquitetura do seu processador (32-bit ou 64-bit no caso do Windows).

Faça o download

Selecione a arquitetura do seu processador e clique, o download será iniciado automaticamente. Caso não saiba, vá em Configurações > Sistema > Sobre > Tipo de Sistema e veja qual a arquitetura.

Instale

Termine a configuração selecionando o repósitorio de destino dos arquvio e clique em instalar para finalizar o processo.

Windows: Abra o instalador `.exe` baixado e siga as instruções na tela. Mantenha as opções padrão a menos que você tenha alguma configuração específica.

MacOS: O Git pode ser instalado via o instalador `.dmg` baixado ou via Homebrew (usando o comando `brew install git` no terminal).

Linux:No terminal, execute `sudo apt-get install git` (Debian/Ubuntu) ou `sudo yum install git` (CentOS/Fedora).

Verifique a instalação

Abra o Terminal (no macOS/Linux) ou o Prompt de Comando (no Windows).

- Digite o comando abaixo para verificar se o Git foi instalado corretamente: git --version

- Se aparecer a versão do Git (por exemplo, `git version 2.30.0`), a instalação foi bem-sucedida!

Configure o Git localmente

Antes de criar um repositório, vamos configurar o Git com suas informações.

Nome e e-mail

Abra o terminal e digite os comandos abaixo, substituindo `Seu Nome` e `seu-email@example.com` pelas suas informações:

git config --global user.name "Seu Nome"

git config --global user.email "seu-email@example.com"

Essas informações serão associadas às suas alterações nos repositórios.
